## Quick Start

```bash
python3 -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
python ./src/csv_to_mbra/csv_to_mbra_xml.py
```

```powershell
python -m venv venv
.\venv\Scripts\activate.ps1
pip install -r requirements.txt
python ./src/csv_to_mbra/csv_to_mbra_xml.py
```
